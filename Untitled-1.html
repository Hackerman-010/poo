A

Abstração

Selecionar o que é relevante de um objeto e esconder os detalhes irrelevantes. Permite modelar entidades do mundo real em software.

E

Encapsulamento

Ocultar os dados internos (atributos) e expor uma interface (métodos) segura. Exemplos: private, getters e setters.

H

Herança

Permite criar uma classe a partir de outra (extends) e reaproveitar código comum.

P

Polimorfismo

Objetos de tipos diferentes respondem de formas diferentes a mesma chamada de método (overriding / overloading).

Conceitos de POO

Explicações curtas e exemplos conceituais.

Classe e Objeto

Classe é uma planta; objeto é uma instância. Uma classe define atributos (dados) e métodos (comportamentos).

Atributos e Métodos

Atributos representam estado; métodos representam ações. Em Java atributos normalmente têm modificadores (private/protected/public).

Construtores

Métodos especiais que inicializam objetos. Podem existir múltiplos construtores (sobrecarga).

Pacotes (packages)

Organizam classes em namespaces — ajudam na modularidade e controle de acesso.

public class ContaBancaria {
  private double saldo;
  public ContaBancaria() {}
  public ContaBancaria(double saldoInicial) {
    this.saldo = saldoInicial;
  }
  public double getSaldo(){ return saldo; }
  public void depositar(double valor){ saldo += valor; }
}

Java e Ferramentas

O papel do JDK, JVM e ferramentas de desenvolvimento.

JDK

Java Development Kit: inclui compilador (javac), java launcher, jar, e utilitários. Instale e configure JAVA_HOME.

JVM

Máquina virtual que executa bytecode (.class). É a peça que torna o Java "write once, run anywhere".

IDE

IDEs comuns: IntelliJ IDEA, Eclipse, NetBeans — ajudam a compilar, debugar e gerenciar projetos.

JDK contém o JRE e ferramentas como java, javac e jar.

Bytecode & JIT

Código .java compilado para .class (bytecode). A JVM pode usar JIT (Just-In-Time) para otimizar performance em runtime.

Tipos de Dados & Operadores

Primitivos, inteiros e operadores usados em Java.

Tipos primitivos

  • boolean — true/false
  • byte — 8 bits
  • short — 16 bits
  • int — 32 bits
  • long — 64 bits
  • float / double — ponto flutuante
  • char — caractere

Operadores

Aritméticos: + - * / % ++ --

Relacionais: == != > < >= <=

Lógicos: && || ! (curto-circuito) e & | ^

Uso: operadores usados em condicionais e laços para controlar fluxo.

Atividade: Máquina de Café (esquema)

Implemente uma classe Cafeteira com atributos de ingredientes (água, café, leite, capacidade) e métodos para servir(), repor() (com verificação de não exceder capacidade) e mostrarStatus().

// ideia de esquema em Java
public class Cafeteira {
  private int aguaMl;
  private int cafeGr;
  private final int CAPACIDADE_AGUA = 2000;
  public Cafeteira(){ /*...*/ }
  public void reporAgua(int ml){
    aguaMl = Math.min(CAPACIDADE_AGUA, aguaMl + ml);
  }
  // servir, status...
}

Sugestões de teste

  • Testar reposição maior que a capacidade — não deve exceder.
  • Testar servir quando recursos insuficientes — retornar erro/false.
  • Testar construtores com saldo inicial (no caso da ContaBancaria).
